About this Application

TA Hours queue is a database backed web application that tracks teaching hours sessions. Its primary function is to give students and teaching assistants a responsive queue that makes tracking information easier. It also allows the queue to be on the web, which makes it easily accessible (rather than a physical queue on a board in some random classroom). It can also be used to perform data analysis on the data held within the databases, which can give key statistics for funding and educational purposes.

About this Project

This application was created as part of a special studies project by the author in her final semester of her undergraduate career. It was built to tackle some of the issues she saw as a teaching assistant. For more information on this project, see its github repository
